# Application Streamlit + SQLite

Cette application propose une interface simple pour inserer des contacts dans une base SQLite.

## Prerequis

- `uv` installe sur la machine
- Python 3.12+

## Installation des dependances

```bash
uv sync
```

## Lancer l'application

```bash
uv run streamlit run main.py
```

## Base de donnees

Le fichier SQLite est cree automatiquement a la racine du projet sous le nom `app.db`.
